Figma export — Wider Circle CMS

Each link below renders one screen at a fixed 1440 px width, plugin-ready. Recommended flow: html.to.design (inside Figma) → paste any URL → Import. Variants let you capture key states of the same screen.

How to import to Figma
  1. In Figma, install the html.to.design plugin (free tier works for single screens).
  2. Open a Figma file, run the plugin, choose Import from URL.
  3. Copy one of the links below (right-click → Copy link), paste it in, click Import.
  4. Each screen lands as a frame with nested layers — auto-layout, colors, and text styles mostly preserved.
  5. Repeat for every variant. Group them into pages in Figma (e.g. Patient 360 / variants).

Notes for plugin import

  • Each URL renders at exactly 1440 px wide.
  • Modals/overlays use position:fixed — plugins typically flatten these into the top of the frame. Crop/position in Figma after import.
  • Fonts (Montserrat, Inter, Azeret Mono) load from Google Fonts; Figma will map them to the same family names if installed locally.
  • Colors and radii come from CSS variables — importers that support variables may pick these up automatically; otherwise you'll get literal hex values.
  • After import, promote repeated elements (Badge, Button, Avatar, CaseRow, Timeline item) to Figma components to get a proper library.
